# Mobile Smoke Checklist (390x844)

Use a viewport of `390x844` and verify the following:

- Drawer:
  - Open via the mobile header hamburger button.
  - Close via the backdrop.
  - Re-open, then close via the in-drawer close button.
  - Re-open, then close via `Escape`.
- Board switching:
  - Use the `Column switcher` tabs to move between columns.
  - Confirm the horizontal board scroller exists (`Board columns`) and columns snap while scrolling.
- Activity log:
  - Open from the mobile header activity button.
  - Dismiss via backdrop.
  - Re-open, then dismiss via `Escape`.
  - Confirm the activity sheet content area is vertically scrollable.
